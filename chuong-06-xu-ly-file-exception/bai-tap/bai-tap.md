# Bài Tập Chương 6: File I/O & Exception

## Bài 1: Đếm dòng/từ/ký tự (⭐ Dễ)

**Yêu cầu:** Đọc file text, in ra: số dòng, số từ, số ký tự (giống lệnh `wc` Linux).

---

## Bài 2: CSV → JSON (⭐⭐ Trung bình)

**Yêu cầu:** Đọc file CSV sinh viên (ten,tuoi,diem), chuyển sang JSON và ghi ra file mới.

---

## Bài 3: Custom Exception (⭐⭐⭐ Khó)

**Yêu cầu:** Tạo exception `ValidationError`. Viết hàm validate đăng ký: email phải có @, mật khẩu >= 8 ký tự.
