## Yeu Cau

- May tinh co cai dat Python 3.10 tro len
- Kien thuc co ban ve su dung may tinh
- Mot IDE hoac text editor (khuyen nghi VS Code)
- Tinh than hoc hoi va thuc hanh

## Cach Su Dung Giao Trinh

1. Doc tung chuong theo thu tu tu 01 den 09
2. Thuc hanh cac vi du trong thu muc `examples/` cua moi chuong
3. Lam bai tap cuoi moi chuong de cung co kien thuc
4. Lam du an thuc hanh o chuong 09 de tong hop kien thuc

## Moi Truong Hoc Tap Khuyen Nghi

- **Python 3.10+** (khuyen nghi 3.12)
- **VS Code** voi extension Python
- **pip** de quan ly thu vien
- **venv** de tao moi truong ao

## Cai Dat Nhanh

```bash
# Kiem tra phien ban Python
python3 --version

# Tao moi truong ao
python3 -m venv myenv
source myenv/bin/activate  # Linux/Mac
myenv\Scripts\activate     # Windows

# Cai dat cac thu vien can thiet (cho chuong 08-09)
pip install requests pandas flask fastapi pytest
```
