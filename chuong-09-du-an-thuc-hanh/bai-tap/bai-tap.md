# Bài Tập Chương 9: Dự Án Thực Hành

## Dự Án 1: Todo CLI (⭐⭐ Trung bình)

Viết ứng dụng todo trên terminal: add, list, done, delete. Lưu JSON.

---

## Dự Án 2: Phân tích text (⭐⭐ Trung bình)

Đọc file text lớn → đếm từ, tìm từ phổ biến nhất, tính số câu, từ trung bình/câu.

---

## Dự Án 3: Quản lý chi tiêu (⭐⭐⭐ Khó)

CLI app: thêm chi tiêu (ngày, mô tả, số tiền, danh mục), thống kê theo tháng/danh mục. Lưu CSV.
